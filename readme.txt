A simple and nice plugin to add date restrict dynamically in gravity forms default date picker field.

== Description ==
A simple and nice plugin to add date restrict dynamically in gravity forms default date picker field. If the installation is okay, go to gravity forms dashboard. Add a date field on your form. Open date field and you will see to Enable Restrict date options checkbox. Click that checkbox and configure as you want. And save form. You're done!

[View Pro Demo](https://pluginscafe.com/demo/restrict-dates-for-gravity-forms-pro/) | [Documentation](https://pluginscafe.com/docs/restrict-dates-for-gravity-forms/) | [Buy Pro](https://pluginscafe.com/plugin/restrict-dates-for-gravity-forms-pro/)

== Features ==
* Minimum date set to a current and specific date
* Maximum date set to a current and specific date
* Disable manually input (read-only option)
* Week/Off day disable
* Disable specific dates (new)
* Week start day (new)

== Premium Features ==
